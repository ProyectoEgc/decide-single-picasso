IssueType(status): Issue's title
Description 

###Formato de Commit:

#Tipo de commit(estado): Titulo de issue
#Breve descripción

###Tipos de Commit:

#add: se genera una nueva funcionalidad
#fix: se soluciona un bug
#refactor: refactorización y mejoras
#delete: se eliminan funciones y servicios
#docs: se genera cambios de la documentación

###Estados de Commit:
#initial: es el primer commit para esa issue
#update: si se modifica algo pero aun no esta finalizada la issue
#end: se termina la issue
